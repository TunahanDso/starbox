# starbox
StarBox: Web tabanlı, maç bazlı (20–25 dk) 2–4 oyunculu uzay oyunu. Yıldız boşluğunda kare gezegenlere warp at, kaynak topla (metal/ahşap), yapı kur, kuleleri fethet; final hedefi Kuroi. İlerleme: craft + perk. Otoriter sunucu (host PC) + Cloudflare Tunnel, Next.js + Socket.IO + Node. Alfa MVP.
